\# 🏥 Hospital Emergency Room Dashboard
## 📊 Interactive Dashboard

Explore the interactive Tableau dashboard for this project:

[View Hospital Dashboard](https://public.tableau.com/views/hospital_dashboard_17752173336120/finalDashboard?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)



\## 📌 Objective

The objective of this project is to analyze hospital emergency room data to understand patient flow, waiting times, and overall performance. This helps in improving hospital efficiency and patient care.



\---



\## 📁 Dataset

\- File Type: CSV / Excel  

\- Description:  

&#x20; The dataset contains information about emergency room visits including patient count, waiting time, department, and visit trends.



\---



\## 🛠 Tools Used

\- Tableau (Data Visualization)

\- Excel / CSV (Data Source)

\- GitHub (Project Hosting)



\---



\## 📊 Dashboard Features

\- Total number of patients

\- Average waiting time

\- Department-wise patient distribution

\- Monthly patient trends

\- Interactive filters (Month, Department, etc.)



\---



\## 📅 Monthly Analysis

This dashboard includes month-wise analysis to identify trends and patterns in emergency room visits.



\### 🔍 Key Monthly Insights

\- Certain months show higher patient inflow (peak periods)

\- Waiting time increases during high patient load

\- Some months have lower activity, indicating better service efficiency

\- Helps in planning hospital resources effectively



\---



\## 🔍 Key Insights

\- Peak patient load occurs in specific months

\- Waiting time is higher in some departments

\- Patient flow varies based on time and demand

\- Resource planning is needed during busy periods
